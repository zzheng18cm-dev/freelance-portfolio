## 可直接粘贴的案例文案

### 案例 1：网页数据采集器
> **一句话**：可配置的 Python 爬虫，自动翻页抓取列表数据，清洗后一键导出 Excel/CSV。
> **技术**：requests + BeautifulSoup + openpyxl，自动翻页、数据清洗、有礼貌的请求间隔、UTF-8 防乱码。
> **成果**：实测稳定抓取 100 条数据并导出。换站点只需改选择器，可扩展代理/并发/入库。

### 案例 2：文件批量整理工具
> **一句话**：把杂乱文件夹按类型/日期自动归类的命令行工具，带预览模式防误操作。
> **技术**：纯 Python 标准库（pathlib/shutil/argparse），dry-run 预览、重名防覆盖。
> **成果**：一条命令整理上百文件，零依赖、任何电脑直接跑。

### 案例 3：本地文档 AI 问答（Mini-RAG）
> **一句话**：把本地文档变成可问答的知识库，AI 基于文档作答并标注来源——企业知识库/智能客服内核。
> **技术**：纯 Python 实现 TF-IDF 检索 + 大模型 API 生成，Prompt 约束防幻觉。
> **成果**：完整 RAG 链路跑通，可扩展向量检索、PDF/Word、网页或微信问答机器人。
