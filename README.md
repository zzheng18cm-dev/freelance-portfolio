## 1. 通用网页数据采集器
可配置的 Python 爬虫：自动翻页抓取列表数据，清洗后导出 Excel / CSV。

- **解决的问题**：从网页批量获取数据（商品、价格、列表信息等）时，人工复制慢、易漏、难维护。
- **实现**：requests 发请求（自定义 UA + 请求间隔，降低被封风险）→ BeautifulSoup 解析 → 数据清洗（价格转浮点、评分文字转数字）→ 自动翻页 → 导出带 BOM 的 CSV（Excel 不乱码）和 openpyxl Excel。
- **亮点**：正则提取价格规避编码干扰、强制 UTF-8 防乱码、模块化解析层让换站成本最小。
- 换目标站点只需改 URL 与选择器；可扩展代理池、并发、增量去重、入库。

## 2. 文件批量整理工具
把杂乱文件夹按类型/日期自动归类的命令行工具。

- **解决的问题**：下载/桌面文件夹堆满杂乱文件，手动整理费时。
- **实现**：纯 Python 标准库（pathlib / shutil / argparse），按扩展名归类到「图片/文档/视频/音频/压缩包/程序/其它」，可选再按修改月份分层。
- **亮点**：`--dry-run` 预览模式防误操作、重名自动加序号绝不覆盖、零第三方依赖。

## 3. 本地文档 AI 问答 (Mini-RAG)
把本地文档变成可问答的知识库，AI 基于文档作答并标注来源——企业知识库 / 智能客服的核心。

- **解决的问题**：文档多、找信息难；直接问通用 AI 又没有内部资料、易胡编。
- **实现**：纯 Python 手写 TF-IDF 检索从文档中找最相关片段 → 作为上下文调用大模型 API 生成有依据的回答；Prompt 约束「只根据资料回答 + 标来源」防幻觉。
- **亮点**：检索部分零重型依赖即可运行；可扩展向量嵌入 + 向量库、PDF/Word、网页或问答机器人。
